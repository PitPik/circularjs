Demo of the demos

This demo shows how the demo app itself is working, so what you're just seeing.

This demo is driven by a navigation that also shows its state and an outlet that shows the pages controlled by the menu, that show the individual demos like this you just see.

To set up this demo we use a navigation component 'demo-nav' with a state controller, a router and a dynamic component loader. As you can see with the attribute cr-lazy components can be lazy loaded.
