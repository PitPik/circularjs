Demo of the demos

This demo shows how the demo app itself is working, so what you're just seeing.

This demo is driven by a navigation that also shows its state and an outlet that shows the pages controlled by the menu, that show the individual demos like this you just see.

To set up this demo we use a navigation component 'demo-nav' with a state controller, a router and a module / widget loader that has plenty more options for transitions, preloading, ...

All the modules are actually individual HTML pages that are loaded via circular.renderModule() into <div class="module-outlet"></div> and can therefor also be called as web-pages/widgets without the surrounding app. Those modules can be developed independently.
Just replace the '#' in the URL with 'demos' and add '.html' at the end of the URL.
